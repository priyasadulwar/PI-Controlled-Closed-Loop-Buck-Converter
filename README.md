# PI-Controlled Closed-Loop Buck Converter

## Project Overview
This project presents the design and simulation of a PI-controlled closed-loop buck converter using MATLAB Simulink. The PI controller regulates the output voltage under varying load conditions and improves the stability and performance of the converter.

## Tools Used
- MATLAB
- Simulink

## Key Features
- Closed-loop voltage control using a PI controller
- Output voltage regulation under varying load conditions
- Analysis of output voltage and current waveforms
- Simulation-based implementation in MATLAB Simulink

## Repository Contents
- PIClosedLoopBuckConverter.slx – Simulink model
- ProjectReport.pdf – Detailed project report
- ProjectPPT.pptx – Project presentation
- Images/ – Circuit diagram and output waveforms
